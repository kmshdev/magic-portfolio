## Introduction

Welcome to my portfolio and technical blog. I'm Keshav Mishra, also known as kmsh, and I build autonomous agents and production-grade LLM systems.

Over the past five years, I've worked across multiple industries—retail, finance, oil and gas, and healthcare—delivering ML solutions that create measurable impact. My focus is on taking cutting-edge AI research and making it work reliably in production environments.

## What I Do

I specialize in autonomous agent engineering. These are systems that can perceive their environment, make decisions, and take actions with minimal human intervention. Think beyond chatbots—I build agents that analyze data, execute trades, optimize workflows, and solve complex problems.

Some highlights from my work include building an LLM-powered analytics agent that reduced turbomachinery repair time by twenty percent, creating agent scaffolding platforms used across the Olas ecosystem, and contributing to autonomous trading systems for prediction markets.

## My Approach

Production ML is fundamentally different from research ML. A model that works in a notebook isn't the same as a system that runs reliably at scale. My approach emphasizes reliability, observability, and gradual deployment. Every system I build includes comprehensive monitoring, fallback mechanisms, and clear success metrics.

I'm a strong believer in using the right tool for the job. My stack includes Python for ML pipelines, TypeScript for agent orchestration, and Rust for performance-critical components. I work with frameworks like LangChain, DSPy, and Haystack, and deploy on infrastructure ranging from AWS to Kubernetes clusters.

## Open Source

I contribute actively to open source projects and maintain several packages. You can find my work on GitHub and PyPI, including genie-cli for agent scaffolding, plugin-memeooorr for meme-centric agents, and various contributions to the Valory ecosystem.

## What to Expect

This blog will cover practical insights from building production ML systems. I'll write about architecture patterns that work, deployment strategies that scale, and lessons learned from real-world projects. Expect technical depth without unnecessary complexity.

Topics I plan to cover include autonomous agent architectures, LLM reliability patterns, MLOps best practices, and case studies from my projects. I'll share both successes and failures, because the failures often teach more than the wins.
