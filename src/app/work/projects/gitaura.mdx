## Overview

GitAura is a developer analytics dashboard that transforms GitHub profiles into stunning visualizations. It features contribution graphs, an Aura Points system that gamifies developer activity, global and monthly leaderboards, and a comprehensive achievement badge system.

## Key Features

- **Beautiful Visualizations**: Interactive contribution graphs with multiple themes (Light, Dark, Ocean)
- **Aura Points System**: Gamified scoring system that rates developer activity across commits, PRs, issues, and code reviews
- **Leaderboards**: Global and monthly rankings to discover top contributors
- **Achievement Badges**: Unlock badges for milestones and special contributions
- **Profile Embedding**: Embed your dashboard in your GitHub README
- **Multiple Themes**: Customizable appearance to match personal branding

## Technologies Used

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **GitHub API**: Profile and contribution data
- **Recharts**: Data visualization library
- **TailwindCSS**: Styling and theming
- **Vercel**: Deployment and hosting

## Aura Points Algorithm

The Aura Points system evaluates developer activity across multiple dimensions:
- **Commits**: Weighted by repository stars and impact
- **Pull Requests**: Scored by complexity and review feedback
- **Issues**: Valued by engagement and resolution
- **Code Reviews**: Rated by thoroughness and helpfulness
- **Consistency**: Bonus points for regular contributions

## Impact

- **Developer Engagement**: Gamification encourages consistent open-source contributions
- **Profile Enhancement**: Developers use GitAura dashboards in portfolios and READMEs
- **Community Building**: Leaderboards foster friendly competition and discovery
- **Analytics Insights**: Developers gain insights into their contribution patterns

## Challenges and Learnings

The primary challenge was designing a fair and meaningful scoring system that couldn't be easily gamed. This required careful weighting of different contribution types and implementing anti-gaming measures. Additionally, optimizing GitHub API usage to stay within rate limits while providing real-time data required intelligent caching strategies.

## Features in Development

- **Team Analytics**: Aggregate dashboards for organizations
- **Contribution Streaks**: Track and celebrate consistency
- **Language Breakdown**: Visualize programming language usage
- **Repository Impact**: Measure influence of individual repositories
- **AI Insights**: LLM-powered analysis of contribution patterns

## Outcome

GitAura has become a popular tool for developers looking to showcase their GitHub activity in a visually appealing way. It demonstrates the power of data visualization and gamification in developer tools.
